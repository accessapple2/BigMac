## Root Cause (for future reference)

`initialize_arena()` in main.py hardcoded all 18+ agents to `url=OLLAMA_URL` (localhost) as part of the "RAM patch 2026-04-17". This funneled every war_room inference to bigmac's local Ollama, where `qwen3.5:9b` (8GB) + Python trader (~560MB) + OS overhead (~2-3GB) = 15GB on a 16GB box. Any inference triggered swapping → 120s timeouts → cascading war_room failures.

**Fix:** Import `OLLIE_URL` in `initialize_arena()`, route all GPU-capable agents to Ollie (RTX 5060, 192.168.1.166:11434).
